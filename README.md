This repository cotains code for analyzing incentive effects on an evidence integration task. 

This code has been developed after collecting data for a discovery sample (exp1), with the purpose of pre-registering the main analyses before collecting a replication sample (exp1rep). 


To use the source code required for analyses, first do:

&nbsp;

pip install -e .

Pre-registered analyses can be run with 

`python scripts/prereg/run_prereg_pipeline.py --exp-name exp1`
`python scripts/prereg/run_prereg_pipeline.py --exp-name exp1rep`
